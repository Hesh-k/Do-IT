# Do-IT

This is a small weekend project created to learn about basic CRUD operations. It is a simple to-do list application that allows users to manage their tasks.

## Features

- Add new tasks
- Edit existing tasks
- Mark tasks as completed
- Delete tasks
- View all tasks

## Technologies Used

- HTML
- PHP
- CSS
- JavaScript
- SQL
