# Do-IT

This is a small weekend project that created to learn about dynamic content creation using js. It is a simple to-do list application that allows users to manage their tasks.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- View all tasks

## Technologies Used

- HTML
- CSS
- JavaScript
